# 📝 글 근육 습관 추적기

> 프로작가를 향한 여정 - 글 근육을 키우는 점진적 훈련 시스템

소설가를 위한 체계적인 글쓰기 습관 관리 도구입니다. 하루에 여러 번 나눠서 쓰는 소설 작업을 세밀하게 추적하고, 단계적으로 글쓰기 체력을 늘려가는 레벨 시스템을 제공합니다.

## ✨ 주요 기능

### 🎯 **세션 기반 글쓰기 추적**
- **여러 세션 지원**: 하루에 몇 번이든 글쓰기 세션 추가 가능
- **실시간 총합**: 모든 세션의 시간과 글자 수 자동 집계
- **시간대 표시**: 각 세션이 언제 작성되었는지 기록
- **세션 관리**: 개별 세션 수정/삭제 기능

### 📊 **레벨 시스템**
- **11단계 레벨**: 초보 작가(5분)부터 마라톤 작가(4시간)까지
- **점진적 성장**: 연속 달성 일수에 따른 자동 레벨업
- **맞춤형 목표**: 현재 레벨에 맞는 일일 목표 시간 설정

### 📈 **상세한 분석 및 통계**
- **주간 통계**: 이번 주 총 글쓰기 시간 및 글자 수
- **성장 지표**: 분당 평균 글자 수, 최고 기록, 최장 연속 등
- **누적 추적**: 총 누적 글자 수 및 전체 글쓰기 시간
- **코치 조언**: AI 기반 개인화된 동기부여 메시지

### 🏃‍♂️ **보조 습관 관리**
- **운동**: 몸의 체력 = 글의 체력
- **일본어**: 언어의 다양성 확장
- **블로그**: 글쓰기 실전 연습

## 🚀 시작하기

### 설치 및 실행
1. HTML 파일을 다운로드하여 웹 브라우저에서 열기
2. 별도 설치 과정 없이 바로 사용 가능
3. 모든 데이터는 브라우저 로컬 스토리지에 자동 저장

### 첫 사용법
1. **세션 추가**: 시간과 글자 수를 입력하고 "세션 추가" 클릭
2. **목표 달성**: 현재 레벨의 목표 시간을 채우고 "목표 달성 완료" 클릭
3. **보조 습관**: 운동, 일본어, 블로그 버튼을 클릭하여 완료 표시

## 📖 사용 가이드

### 세션 관리
```
📖 오늘의 소설 쓰기
┌─────────────────────────────┐
│ 09:30 세션: 15분, 280자     │ ✏️ ✕
│ 21:00 세션: 25분, 420자     │ ✏️ ✕
│ + 세션 추가                 │
└─────────────────────────────┘
총합: 40분, 700자 ✅ 목표 달성!
```

#### 세션 추가
- 시간(분): 1-480분 (최대 8시간)
- 글자 수: 0-50,000자 (선택사항)
- **단축키**: `Ctrl + Enter` 또는 `Cmd + Enter`

#### 세션 편집
- ✏️ 버튼 클릭하여 시간과 글자 수 수정
- ✕ 버튼 클릭하여 세션 삭제 (확인 대화상자 표시)

### FAQ

**Q: 데이터가 사라질 위험이 있나요?**
A: 브라우저 데이터 삭제나 시스템 문제로 사라질 수 있으므로 정기적인 백업을 권장합니다.

A: 현재는 로컬 저장만 지원합니다. 데이터 다운로드/업로드 기능으로 수동 동기화 가능합니다.

**Q: 목표 시간을 변경할 수 있나요?**
A: 현재는 레벨별 고정 목표를 사용합니다. v3.0에서 커스터마이징 기능 추가 예정입니다.
