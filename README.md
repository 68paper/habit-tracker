# 📝 글 근육 습관 추적기 v3.0

프로작가를 향한 여정 - 글 근육을 키우는 점진적 훈련 시스템

## 📌 개요

소설가를 위한 체계적인 글쓰기 습관 관리 도구입니다. 하루에 여러 번 나눠서 쓰는 소설 작업을 세밀하게 추적하고, 단계적으로 글쓰기 체력을 늘려가는 레벨 시스템을 제공합니다.

### 핵심 철학
- **점진적 성장**: 5분부터 시작해서 4시간까지 단계별로 성장
- **세션 기반 추적**: 하루 중 여러 번 나눠서 쓰는 패턴 지원
- **개인화된 습관 관리**: 나만의 습관을 자유롭게 추가하고 관리
- **지속 가능한 동기부여**: 연속일 기록과 보호막 시스템으로 부담 없는 습관 형성

## ✨ 주요 기능

### 🔥 1. 연속일 기록 시스템
- **글쓰기 연속일**: 매일 목표 달성시 연속 기록 증가
- **운동 연속일**: 운동 기록시 연속일 카운트
- **전체 활동 연속일**: 모든 습관을 포괄하는 종합 연속 기록
- **시각적 피드백**: 불꽃 애니메이션과 함께 성취감 극대화

### ❄️ 2. 연속일 보호막 시스템
- **기본 보호막 3개**: 하루 놓쳐도 연속 기록 보호
- **자동 충전**: 주간 목표 달성시 보호막 1개 자동 충전
  - 글쓰기 주 5일 달성 또는
  - 전체 활동 주 6일 달성
- **최대 5개 보유**: 과도한 축적 방지로 적절한 긴장감 유지
- **유연한 사용**: 컨디션 난조일이나 특별한 상황에 사용

### 🎯 3. 사용자 정의 습관 시스템 (v3.0 신규)
- **자유로운 습관 추가**: 나만의 습관을 무제한 생성
- **두 가지 타입 지원**:
  - **단순 체크**: 완료/미완료 방식 (예: 독서, 명상)
  - **레벨 시스템**: 횟수 기반 성장 (예: 운동, 언어학습)
- **색상 지정**: 8가지 색상으로 습관 구분
- **완전한 관리**: 습관 추가, 수정, 삭제 자유자재
- **기본 습관 포함**: 일본어, 블로그 습관이 기본 제공

### 4. 세션 기반 글쓰기 추적
- **다중 세션 관리**: 하루에 여러 번 글쓰기 세션 기록
- **실시간 집계**: 모든 세션의 시간과 글자 수 자동 합산
- **세션별 관리**: 개별 세션 수정/삭제 기능
- **시간대 기록**: 각 세션의 작성 시각 표시

### 6. 상세 통계 및 분석
- **주간 통계**: 이번 주 총 글쓰기 시간 및 글자 수
- **일평균 계산**: 평균 글쓰기 시간과 글자 수
- **성장 지표**: 
  - 분당 평균 글자 수
  - 일일 최고 기록
  - 최장 연속 기록
  - 총 누적 글자 수
- **개인화된 코치**: 상황에 맞는 동기부여 메시지

## 🚀 사용 방법

### 설치
1. HTML 파일을 다운로드
2. 웹 브라우저에서 파일 열기
3. 별도 설치 없이 바로 사용 가능
4. 모든 데이터는 브라우저에 자동 저장

### 기본 사용법

#### 1. 글쓰기 세션 추가
```
시간: 25분
글자: 450자
[+ 세션 추가] 클릭 또는 Ctrl+Enter
```

#### 2. 목표 달성
- 현재 레벨의 목표 시간 달성
- "목표 달성 완료하기" 버튼 활성화
- 클릭하여 오늘 완료 표시

#### 3. 운동 기록
```
시간: 30분
소모 칼로리: 250kcal
[+ 운동 기록 추가] 클릭
```

#### 4. 사용자 정의 습관 관리 (v3.0)
- **습관 추가**: "+ 새 습관 추가" 버튼 클릭
- **단순 체크형**: 완료/미완료만 체크
- **레벨형**: 목표 횟수 설정 후 "+1" 버튼으로 기록
- **습관 삭제**: 각 습관의 "✕" 버튼으로 삭제

#### 5. 연속일 기록 관리
- 매일 목표 달성시 자동으로 연속일 증가
- 연속일 보호막으로 하루 정도는 안전하게 휴식 가능
- 주간 목표 달성시 보호막 자동 충전

### 날짜 이동
- ← 이전 / 다음 → 버튼으로 날짜 이동
- "오늘로 이동" 버튼으로 현재 날짜로 복귀

## 🎯 사용자 정의 습관 활용법

### 추천 습관 예시

#### 단순 체크형
- 📚 독서 (30분)
- 🧘 명상 (10분)
- 🚶 산책 (20분)
- 📱 스마트폰 금식
- 🛏️ 일찍 잠자리

#### 레벨 시스템형

- **파란색 (blue)**: 학습 관련 습관

- **보라색 (purple)**: 창작 관련 습관
- **빨간색 (red)**: 중요도가 높은 습관
- **노란색 (yellow)**: 재미있는 활동
- **분홍색 (pink)**: 자기계발
- **인디고 (indigo)**: 정신적 활동

## 🛡️ 연속일 보호막 전략

### 보호막 활용법
1. **컨디션 난조일**: 몸이 아프거나 특별한 일이 있을 때
2. **예상 가능한 휴식**: 여행이나 중요한 일정이 있을 때
3. **번아웃 방지**: 너무 무리하지 않고 지속 가능한 습관 유지

### 보호막 충전 조건
- **글쓰기 중심**: 주 5일 이상 글쓰기 목표 달성
- **전체 활동**: 글쓰기 + 운동 + 사용자 정의 습관으로 주 6일 이상 활동

### 권장 전략
- 보호막은 정말 필요할 때만 사용
- 주간 목표를 통해 꾸준히 충전
- 최대 5개까지만 보유 가능하니 적절히 사용

### 보호막 사용 시나리오
```

화: 글쓰기 25분 + 명상 ✅ (연속 6일)  
수: 감기로 휴식 ❄️ 보호막 사용 (연속 7일 유지)

```

### 주간 목표 예시
- **최소 목표**: 글쓰기 주 3회 + 다른 습관 주 2회 = 주 5회 활동
- **표준 목표**: 글쓰기 주 5회 + 다른 습관들 = 주 8회 활동  
- **챌린지 목표**: 매일 무언가 하나씩 = 주 7회 활동

## 🔒 개인정보 보호

### 데이터 보안
- ✅ 모든 데이터는 로컬 브라우저에만 저장
- ✅ 외부 서버 전송 없음
- ✅ 오프라인 사용 가능
- ✅ 완전한 데이터 소유권

### 백업 권장사항
- 주 1회 정기 백업
- 중요 마일스톤 달성 시 백업
- 새로운 습관 추가 후 백업
- 브라우저 데이터 삭제 전 필수 백업

## 🐛 문제 해결

### 한글이 깨져 보일 때
- 파일을 UTF-8 인코딩으로 저장
- 브라우저 인코딩 설정 확인

### 데이터가 저장되지 않을 때
- 브라우저 LocalStorage 활성화 확인
- 시크릿 모드에서는 저장 안됨
- 저장 공간 부족 확인

### 레벨이 올라가지 않을 때
- 연속 달성 일수 확인 (보호막 사용일 포함)
- 목표 시간 달성 후 "완료" 버튼 클릭 필수

### 연속일이 끊어진 것 같을 때
- 보호막 사용 여부 확인
- 날짜별로 완료 상태 점검
- 필요시 보호막 사용 고려

### 사용자 정의 습관이 안 보일 때
- 페이지 새로고침 시도
- 브라우저 개발자 도구에서 LocalStorage 확인
- 데이터 백업 후 다시 가져오기
