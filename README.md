# 📝 글 근육 습관 추적기

프로작가를 향한 여정 - 글 근육을 키우는 점진적 훈련 시스템

## 📌 개요

소설가를 위한 체계적인 글쓰기 습관 관리 도구입니다. 하루에 여러 번 나눠서 쓰는 소설 작업을 세밀하게 추적하고, 단계적으로 글쓰기 체력을 늘려가는 레벨 시스템을 제공합니다.

### 핵심 철학
- **점진적 성장**: 5분부터 시작해서 4시간까지 단계별로 성장
- **세션 기반 추적**: 하루 중 여러 번 나눠서 쓰는 패턴 지원
- **통합적 관리**: 글쓰기, 운동, 언어학습, 블로깅을 함께 관리

## ✨ 주요 기능

### 1. 세션 기반 글쓰기 추적
- **다중 세션 관리**: 하루에 여러 번 글쓰기 세션 기록
- **실시간 집계**: 모든 세션의 시간과 글자 수 자동 합산
- **세션별 관리**: 개별 세션 수정/삭제 기능
- **시간대 기록**: 각 세션의 작성 시각 표시

### 2. 레벨 시스템

### 3. 상세 통계 및 분석
- **주간 통계**: 이번 주 총 글쓰기 시간 및 글자 수
- **일평균 계산**: 평균 글쓰기 시간과 글자 수
- **성장 지표**: 
  - 분당 평균 글자 수
  - 일일 최고 기록
  - 최장 연속 기록
  - 총 누적 글자 수
- **코치**: 개인화된 동기부여 메시지

### 4. 보조 습관 관리
- **운동**: 몸의 체력 = 글의 체력
- **일본어**: 언어의 다양성 확장
- **블로그**: 글쓰기 실전 연습

## 🚀 사용 방법

### 설치
1. HTML 파일을 다운로드
2. 웹 브라우저에서 파일 열기
3. 별도 설치 없이 바로 사용 가능
4. 모든 데이터는 브라우저에 자동 저장

### 기본 사용법

#### 1. 글쓰기 세션 추가
```
시간: 25분
글자: 450자
[+ 세션 추가] 클릭 또는 Ctrl+Enter
```

#### 2. 목표 달성
- 현재 레벨의 목표 시간 달성
- "목표 달성 완료하기" 버튼 활성화
- 클릭하여 오늘 완료 표시

#### 3. 보조 습관 체크
- 각 습관 버튼 클릭으로 완료 표시
- 운동은 세션 기반으로 상세 기록 가능

### 날짜 이동
- ← 이전 / 다음 → 버튼으로 날짜 이동
- "오늘로 이동" 버튼으로 현재 날짜로 복귀

## 💾 데이터 관리

### 백업 (내보내기)
1. "📥 데이터 다운로드" 클릭
2. JSON 파일로 저장
3. 파일명: `writing_habits_YYYY-MM-DD.json`

### 복원 (가져오기)
1. "📤 데이터 업로드" 클릭
2. 백업 파일 선택
3. 자동으로 데이터 병합 및 변환

### 데이터 호환성
- v1.0 데이터 자동 변환 지원
- 이전 형식 자동 마이그레이션
- 데이터 무결성 자동 검증

- **브라우저**: Chrome, Firefox, Safari, Edge (최신 버전)

- **Frontend**: HTML5, JavaScript (ES6+)

- **저장소**: LocalStorage API
- **인코딩**: UTF-8

## 🐛 문제 해결

### 한글이 깨져 보일 때
- 파일을 UTF-8 인코딩으로 저장
- 브라우저 인코딩 설정 확인

### 데이터가 저장되지 않을 때
- 브라우저 LocalStorage 활성화 확인
- 시크릿 모드에서는 저장 안됨
- 저장 공간 부족 확인

### 레벨이 올라가지 않을 때
- 연속 달성 일수 확인
- 목표 시간 달성 후 "완료" 버튼 클릭 필수
